
--------- Install package ------

npm install express,
npm i nodemon --save-dev,
npx prisma init,
npm i prisma --save-dev,
npm i @prisma/client,
npm i dotenv,
npm i bcryptjs,
npm i jsonwebtoken



--------- prisma migrate -------
npx prisma migrate dev --name add users table
npx prisma migrate dev --name added_others_tables
npx prisma generate

Create server.js file into src folder,

npm run dev
